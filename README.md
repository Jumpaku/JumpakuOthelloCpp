## How to use 使い方
### 
  1. "DOWNLOAD\_ME.zip"をダウンロード
  2. ダウンロードしたファイルを展開
  3. 展開したファイルの中のIncludeフォルダの中身とLibrary内のJumpakuOthello.libをプロジェクトに追加

### 
  1. Download "DOWNLOAD\_ME.zip"
  2. Extract that
  3. Add Library/JumpakuOthello.lib and files in Include to your project
